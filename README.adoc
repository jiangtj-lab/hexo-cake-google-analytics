You can config by `{config_key}` in `hexo` `theme` or `data` config files.

In hexo or theme:

[source,yaml,subs="normal"]
----

include::default.yaml[indent=2]
----

In `{config_key}.yml` data file:

[source,yaml,subs="normal"]
----

----
